```text
6. 하단 보드의 나머지 유닛은 직접 공격하지 않고 머지 재료로 남는다. 보드 전체 DPS 합산 방식은 MVP 기본 규칙에서 사용하지 않는다. 단, 보유 보너스는 MVP 보완 후보로 분리해 구현 부담과 밸런스 검증 후 적용 여부를 결정한다.
```


### 3.4 28.4 보유 보너스 규칙 서두

```text
하단 6x4 보드는 전투 배치판이 아니라 성장 조작판이다. 다만 레퍼런스 분석 결과, 보드의 유닛이 전투와 완전히 분리되면 플레이어가 보드 조작의 전투 의미를 약하게 느낄 수 있다. 따라서 보유 보너스는 MVP 기본 기능이 아니라 MVP 보완 후보로 둔다. 일정과 밸런스 검증이 가능할 때만 제한적으로 적용한다.
```

### 3.5 22.3 스토어 긴 설명 초안

```text
작은 물방울 하나가 세상을 지킬 수 있을까요?

물 키우기에서는 귀여운 물방울을 소환하고, 같은 단계의 물방울을 합쳐 더 강한 물정령으로 진화시킬 수 있습니다.

진화한 물정령은 상단 전투 영역에 직접 출전해 불꽃 병사와 보스를 자동으로 공격합니다.

소환하고, 합치고, 진화시키고, 보상을 모아 더 강해지세요.

짧은 시간에도 성장하는 재미를 느낄 수 있는 캐주얼 머지 디펜스 게임입니다.
```

```text
본 프로젝트는 작은 물방울 캐릭터를 소환하고 같은 단계끼리 합쳐 더 강한 물정령으로 진화시키는 캐주얼 머지 디펜스 게임 기획 사례입니다.
```

## 4. 구조 개선 권장

1. 28장은 “최종 기준”이라기보다 “레퍼런스 반영 보완안” 성격이 강합니다. 문서 상단의 0장과 충돌하지 않도록 28장 제목을 아래처럼 바꾸는 것을 권장합니다.

```text
28. 레퍼런스 기반 보완안 및 MVP 적용 우선순위
```

2. MVP 범위는 아래 3단계로 나누면 개발자가 바로 판단하기 쉽습니다.

```text
MVP 필수: 6x4 보드, 소환, 머지, 최고 등급 1체 대표 출전, 자동 전투, 챕터 1, 캐릭터 5단계, 일반 적 3종, 보스 1종, 기본 강화, 저장
MVP 보완: 일일 미션, 초보자 미션, 오프라인 보상, 보상형 광고, 보유 보너스
MVP 제외: 자동 머지, 보드 정렬 자동화, 시즌 패스, 길드, PvP, 복잡한 장비/룬/유물
```

```text
보스: 챕터 마지막 강적

대상: 공격/스킬이 적용되는 객체
대표 출전 유닛: 보드 내 최고 등급 물정령 1체
```
